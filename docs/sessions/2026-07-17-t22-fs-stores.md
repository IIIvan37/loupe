  mais invalide **throw** (les deux adapters, message partagé) ; `list` le
  **saute** en traçant ; le `load` de préservation de `createdAt` dans
  `saveProject` est **best-effort** — écraser répare.
- **GC = sweep de démarrage bloquant les stores** (barrière), 1×/run,
  conservatif, rapport loggé — jamais pendant les opérations.

  (`~/Library/Application Support/dev.iiivan.loupe/{projects,audio}`) —
  la migration depuis `~/.loupe` est T2.4.
- **Une seule instance desktop** (tauri-plugin-single-instance).
- `clampFineTuneCents` : le contrat « corruption = 0 » couvre désormais les
